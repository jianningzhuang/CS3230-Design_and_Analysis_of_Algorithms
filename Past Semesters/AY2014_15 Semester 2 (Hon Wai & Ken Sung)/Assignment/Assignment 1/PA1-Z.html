Task Z

Given two integers, compute and output their sum. The absolute value of numbers in input is less than 2147483640.

There are two test inputs:
Test 1 (10 points): 
1000000000
1000000000
Test 2 (20 points): 
2000000000
2000000000

Try following C++ code to submit: 

#include <iostream>
using namespace std;

int main()
{	int x, y;
	
	cin >>x>>y;
	cout << x+y << endl;
	return 0;
}
Above code will get Result: 
10 Wrong Answer
Try to guess why it's wrong (?).

Following C++ code will get "Accepted": 
#include <iostream>
using namespace std;

int main()
{	long long x, y;
	
	cin >>x>>y;
	cout << x+y; // without << endl;
	return 0;
}
